## 这里是什么？

**Leon's Group 的内部知识中枢**。我们用它做四件事：

<CardGrid>
  <Card title="📚 一起读论文" icon="open-book">
    每周一篇，带读人在 [共读笔记](#共读笔记) 留下解读；其他人在评论区追问、补充。读后的笔记是**带署名的、活的文档**，不是丢失在群聊里的截图。
  </Card>
  <Card title="🧭 沉淀研究主线" icon="seti:plan">
    [研究主线](/themes/) 是组的研究地图：我们关心什么、做过什么、还有哪些开放问题。新人 30 分钟看完就知道组在干嘛。
  </Card>
  <Card title="👥 看见彼此在想什么" icon="seti:default">
    每个 [成员](/members/) 有自己的笔记区。读到一半的想法、半成品 idea、踩过的坑 —— 写下来全组可见，错过协作机会的概率大幅下降。
  </Card>
  <Card title="🎯 给新人最低成本入门" icon="rocket">
    [新人入口](/onboarding/) 列好"第一周读什么、第一月做什么、三个月做什么"。Leon 不必再口头介绍 100 遍。
  </Card>
</CardGrid>

## 给不同角色的入口

<CardGrid>
  <Card title="🎓 大导师 / 小导师" icon="approve-check">
    在 [研究主线](/themes/) 留观点；在 paper 评论区抛 take；偶尔写一条 research-log 给学生看方向。**写作成本低、权重高**。
  </Card>
  <Card title="🧑‍� 博士生" icon="pencil">
    带读 → 写完整 paper 解读；自己 [research-log](/members/) 持续更新；带师弟时把反复回答的问题写成 wiki。**这里是你的学术 track record**。
  </Card>
  <Card title="📘 硕士生 / 新生" icon="seti:notebook">
    从 [Onboarding](/onboarding/) 开始；学懂一个概念就给 [词典](/concepts/) 加一条；周会后写 100 字短评。**短评一样是贡献**。
  </Card>
  <Card title="🌐 访客 / 合作者" icon="external">
    看 [研究主线](/themes/)、[共读笔记](/deepseek/overview/) 和 [概念词典](/concepts/) 即可了解我们组在做什么。这是组的"公开门面"。
