# Templates

agent 拷贝填空用的原始 markdown 模板。**不是 Astro 内容**（不会被 build），仅供 agent / 人类参考。

| 文件 | 用途 | 推荐生成方式 |
|------|------|-------------|
| `session.md` | 共读 session 三段式 | 优先用 `pnpm new:session`（自动套这个模板）|
| `paper.md` | 论文解读 | 优先用 `pnpm new:paper` |
| `member.md` | 成员主页 | 优先用 `pnpm new:member` |
| `theme.md` | 研究主线 | 没有 scaffold；agent 拷贝本模板 |
| `concept.md` | 概念词条 | 没有 scaffold；agent 拷贝本模板 |

> 这些模板**仅作为 agent fallback** —— 当脚手架命令失效或 agent 想看清"完整字段都有什么"时阅读。
